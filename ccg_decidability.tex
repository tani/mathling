\title{八規則 CCG の導出可能性の決定可能性\\
  \large ——unrestricted type raising を含む体系の決定可能性証明：\\
  Lean 形式化により修正された版（残る補題一つを明示）}
\author{}
\date{}
\maketitle

\begin{abstract}
本稿は，unrestricted type raising を含む八規則 Combinatory Categorial Grammar
(CCG) の導出可能性が判定可能であることの証明を与える．
本版は旧稿（band contraction 版）を Lean 4 による形式化の結果に基づいて
全面的に修正したものである．主な変更点は次の三つ：
(1) 旧稿の中核であった repeatable pair と band contraction の機構は，
trace edge が指し先の部分範疇を保存するという事実により
\textbf{充足不能（空虚に真）}であることが判明したため削除した．
同じ事実から，むしろ強い補題「一つの invisible trace piece は
一本の branch と高々一度しか交わらない」が直ちに従い，
数え上げが大幅に単純化される．
(2) その結果，深さ上界は旧稿の $H = V + qrV(V+1)$ から
$\boxed{H = V + rV}$（$q$ と zone 因子が不要）に鋭化される．
(3) 旧稿の補題「境界なし invisible piece は最小導出に存在しない」の証明
（一様原子置換）は，piece が型上げの骨格 constructor を含む場合に
\textbf{破綻する}ことが反例により判明した．
置換可能な場合は Lean で証明済みである．
骨格を含む場合についても，導出木上の構造帰納法により
「piece が根の二項規則を横断する場合」へ帰着することが Lean で証明済みであり，
その横断ケースだけが本証明に残る唯一の未解決補題である
（\S\ref{sec:protected}）．
この一補題を除き，全段階（原子消去・trace 構造・数え上げ・有限 chart の
健全性/完全性・決定手続き）は Lean で検証済みである．
\end{abstract}

\section*{主定理}
\addcontentsline{toc}{section}{主定理}
%% ============================================================

入力範疇列 $\Gamma = x_1, \ldots, x_n$ と目標範疇 $y$ が与えられたとき，
\[
  x_1, \ldots, x_n \Rightarrow_{\R}^{*} y
\]
が成り立つかどうかは判定可能である．ただし規則集合は
\[
  \R = \{{\mathbin{<}},\,{\mathbin{>}},\,T{\mathbin{<}},\,T{\mathbin{>}},\,B{\mathbin{<}},\,B{\mathbin{>}},\,Bx{\mathbin{<}},\,Bx{\mathbin{>}}\}
\]
である．範疇は $A ::= p \mid A/B \mid A\bs B$ で生成される．

\medskip
\noindent\textbf{証明の大構造．}
証明は四段階からなる．
\begin{enumerate}
  \item \textbf{原子消去（補題~\ref{lem:atom}）}：入力と目標に現れない原子は
    準同型で潰せる．原子集合を有限集合 $\Sig$ に固定できる．
    【Lean 証明済み】
  \item \textbf{境界なし piece の消去（\S\ref{sec:boundary}）}：
    サイズ最小の成功導出において，すべての invisible trace piece は
    visible boundary を持つ．piece が型上げ骨格を含まない場合は
    原子置換により証明済み【Lean 証明済み】．
    骨格を含む場合が\textbf{唯一の未解決補題}（未解決補題~\ref{conj:protected}）．
  \item \textbf{深さ上界（\S\ref{sec:depth}）}：
    境界条件の下で，中間範疇の深さは $H(\Gamma, y) = V + rV$ で有界．
    $V$ は入力と目標から計算でき，$r = 64$ は規則系定数．
    【Lean 証明済み】
  \item \textbf{有限 chart parsing（\S\ref{sec:chart}）}：深さ有界より候補範疇集合
    $K_{\Gamma,y}$ が有限．chart algorithm は停止し，健全．
    完全性は段階 2 の未解決補題のみに条件付けられる．
    【健全性・有界完全性とも Lean 証明済み】
\end{enumerate}

\begin{remark}
旧稿はここに第五の段階として band contraction
（repeatable pair の縮約）を置いていた．
\S\ref{sec:no-revisit} で示すとおり，repeatable pair は定義からして
存在し得ないため，この段階は空虚に真であり，不要である．
\end{remark}

型上げ規則 $T{>},\,T{<}$ における $C$ は\textbf{任意の範疇}である．
これが unrestricted type raising であり，中間範疇が無限に深くなりうる原因である．

\begin{lemma}[二項規則は深さを増やさない]
\label{lem:binary-depth}
六つの二項規則のいずれについても，結論の深さは前提の深さの最大値以下である：
$d(\text{結論}) \le \max(d(\text{左前提}), d(\text{右前提}))$．
\end{lemma}

\begin{proof}
たとえば $B{>}$（$C/B,\,B/A \Rightarrow C/A$）では
$d(C/A) = 1 + \max(d(C), d(A)) \le 1 + \max(d(C/B)-1,\, d(B/A)-1)
= \max(d(C/B), d(B/A))$．他の規則も同様．
\end{proof}

したがって深さを増やす唯一の規則は型上げであり，
決定可能性の困難はすべて「型上げの標的 $C$ をどう有界化するか」に集約される．

\section{導出図}

\textbf{導出図} $D$ とは，有限木であって，各節点に範疇がラベルされ，
各内部節点が八規則のいずれかのインスタンスであるものをいう．
葉は左から順に $x_1, \ldots, x_n$，根は目標範疇 $y$ である．
二項規則は隣接する二区間を結合する．入力長 $n$ の導出では二項規則の使用回数は
常に $n-1$ である【Lean: \lean{binaryRuleCount\_eq\_length\_sub\_one}】．

\begin{verifybox}
\textbf{【例1：具体的な成功導出】}
入力 $a,\;(a\bs s)/b,\;b$ から目標 $s$ への導出（$T{>}$ + $B{>}$ + $>$）：

\medskip
\begin{center}
\begin{prooftree}
  \hypo{a}
  \infer1[$(T{>},\;C{=}s)$]{s/(a\bs s)}
  \hypo{(a\bs s)/b}
  \infer2[$(B{>})$]{s/b}
  \hypo{b}
  \infer2[$(>)$]{s}
\end{prooftree}
\end{center}
\end{verifybox}

\begin{verifybox}
\textbf{【例1$'$：言語学的な例 ``John loves Mary'' と spurious ambiguity】}
語彙を $\text{John}=np$，$\text{loves}=(np\bs s)/np$，$\text{Mary}=np$，
目標 $s$ とする．同じ入力列に対して，型上げを使わない導出（A）と
主語型上げ＋合成を使う導出（B）の\textbf{二通り}が存在する．

\medskip
\begin{center}
\begin{tabular}{cc}
\textbf{導出A（型上げなし）} & \textbf{導出B（主語型上げ＋合成）} \\[4pt]
\begin{prooftree}
  \hypo{np}
  \hypo{(np\bs s)/np}
  \hypo{np}
  \infer2[$(>)$]{np\bs s}
  \infer2[$(<)$]{s}
\end{prooftree}
&
\begin{prooftree}
  \hypo{np}
  \infer1[$(T{>})$]{s/(np\bs s)}
  \hypo{(np\bs s)/np}
  \infer2[$(B{>})$]{s/np}
  \hypo{np}
  \infer2[$(>)$]{s}
\end{prooftree}
\end{tabular}
\end{center}

\smallskip
導出 B では型上げが中間範疇 $s/(np\bs s)$ を生む．この種の「同一目標への複数経路」が
unrestricted type raising の下では無限に増えうるため，深さの有界化が必要になる．
\end{verifybox}

\section{原子消去補題}
\label{sec:atom}

\begin{definition}
入力と目標に現れる原子の集合を
$\Sig = \At(x_1) \cup \cdots \cup \At(x_n) \cup \At(y)$
と定める．
\end{definition}

\begin{lemma}[原子消去補題]
\label{lem:atom}
$x_1, \ldots, x_n \Rightarrow_{\R}^{*} y$ が成り立つなら，途中に現れる原子が
すべて $\Sig$ に属する導出が存在する．
【Lean: \lean{Derivable.toAtomRestrictedDerivableUsingProblemAtoms}】
\end{lemma}

\begin{proof}
$\Sig$ から原子 $p_\ast$ を一つ選び，原子上の写像 $h$ を
\[
  h(p) = \begin{cases} p & p \in \Sig,\\ p_\ast & p \notin \Sig \end{cases}
\]
と定め，$h(A/B) = h(A)/h(B)$，$h(A\bs B) = h(A)\bs h(B)$ と準同型拡張する．
各規則はスキーム規則なので，規則適用が成り立つなら $h$ をかけたものも
同じ規則適用である．$h(x_i)=x_i$，$h(y)=y$ なので，導出全体に $h$ をかければ
所望の導出が得られる．
\end{proof}

以後，原子集合は有限集合 $\Sig$ に固定する．

\section{occurrence と trace edge}
\label{sec:trace}

範疇を二分木と見る．範疇 $A$ の各 $/$・$\bs$ の出現を
\textbf{constructor occurrence} と呼び，その個数を $c(A)$，
深さを $d(A)$ と書く（$d(p)=0$，$d(A/B)=d(A\bs B)=1+\max(d(A),d(B))$）．
位置は $\{0,1\}$ の有限列（根 $\epsilon$，左子 $0$，右子 $1$），
位置 $s$ の部分範疇を $A|_s$ と書く．
導出図 $D$ の節点 $v$ に範疇 $A_v$ があるとき，
\textbf{導出中の constructor occurrence} とは組 $(v,s)$ である．

各規則スキームの $A,B,C$ をメタ変数と呼ぶ．
規則インスタンス $\rho$ におけるメタ変数 $X$ の各出現を
\textbf{メタ変数コピー}と呼ぶ．

\begin{definition}[trace edge]
導出図 $D$ の二 occurrence $(v,s)$，$(w,t)$ の間に \textbf{trace edge} を張るのは，
ある規則インスタンス $\rho$ と同じメタ変数 $X$ の二コピーがあり，
$(v,s)$ と $(w,t)$ がそれぞれその二コピー内部の\textbf{同じ相対位置}にある場合である．
\end{definition}

\textbf{Trace graph} は頂点集合 $\Occ(D)$，辺集合をすべての trace edge とする
無向グラフである．

\medskip
\noindent\textbf{設計判断．}
型上げ $A\Rightarrow C/(A\bs C)$ が\textbf{新しく導入した}外側 $/$ と内側 $\bs$
自体（\textbf{型上げ骨格}と呼ぶ）は，型上げインスタンスの
どのメタ変数コピーの内部 occurrence でもないので，
その型上げインスタンスからは trace edge を持たない
（上位の規則のメタ変数コピー内部としては辺を持ちうる）．

\section{部分範疇保存と branch 非再訪}
\label{sec:no-revisit}

以下の観察が本証明の単純化の鍵である．旧稿はこれを見落としていた．

\begin{lemma}[部分範疇保存]
\label{lem:same-sub}
$(v,s)\sim(w,t)$ が trace edge なら $A_v|_s = A_w|_t$（範疇として同一）．
したがって trace graph の一つの連結成分に属するすべての occurrence は
同一の部分範疇を指す．
【Lean: \lean{SameInvisibleTraceComponent} の \lean{sameSub}】
\end{lemma}

\begin{proof}
trace edge は同じメタ変数 $X$ の二コピーの同じ相対位置を結ぶ．
両コピーには同じ範疇 $\sigma_\rho(X)$ が代入されているから，
指し先の部分範疇は同一である．連結成分については辺に沿って推移的に従う．
\end{proof}

\begin{corollary}[branch 非再訪]
\label{cor:no-revisit}
同一節点 $v$ の同一 branch 上の二 occurrence $(v,s)$，$(v,t)$（$s \prec t$，
$s \neq t$）は，trace graph の同じ連結成分に属さない．
【Lean: \lean{sameInvisibleTraceComponent\_forbids\_strictPrefix\_branch}】
\end{corollary}

\begin{proof}
$s \prec t$ より $A_v|_t$ は $A_v|_s$ の真部分範疇であり，
$c(A_v|_t) < c(A_v|_s)$．補題~\ref{lem:same-sub} より同じ成分なら
$A_v|_s = A_v|_t$ となり矛盾．
\end{proof}

\begin{remark}[旧稿の band contraction 機構は空虚である]
旧稿は「repeatable pair」を
「同一 branch 上の二 invisible occurrence であって，
\emph{同じ invisible trace piece に属し}，同じ interface state を持ち，
間に visible がないもの」と定義し，
その縮約（band contraction）を証明の中核に置いていた．
しかし系~\ref{cor:no-revisit} により，
\textbf{同一 branch 上の二 occurrence が同じ piece に属することはそもそもない}．
すなわち repeatable pair の定義は充足不能であり，
「$\mu$-最小導出に repeatable pair は存在しない」という旧稿の系は
空虚に真，band contraction 補題群は不要である．
この事実は Lean 形式化の過程で発見された．
なお，これは数え上げにとって朗報である：
piece の一致だけで branch 上の再訪が禁止されるため，
interface state（旧稿の $q\approx 2500$ 状態）も
visible-free zone への分割も一切不要になる（\S\ref{sec:depth}）．
\end{remark}

\section{Visible occurrence}
\label{sec:visible}

\begin{definition}[visible / invisible]
導出 occurrence $(v,s)$ が \textbf{visible} であるとは次のいずれかが成り立つこと：
\begin{enumerate}
  \item $v$ が葉である．
  \item $v$ が根である．
  \item $(v,s)$ が二項規則の \textbf{principal constructor} である（二項規則が直接見る最外スラッシュ）．
\end{enumerate}
Application $C/B,B\Rightarrow C$ では左前提の根 $/$；
$A,A\bs C\Rightarrow C$ では右前提の根 $\bs$；
Composition $C/B,B/A\Rightarrow C/A$ では左・右前提・結論の根 $/$，計3個．
型上げが導入した constructor は，それだけでは visible ではない．
visible でない occurrence を \textbf{invisible} と呼ぶ．
\end{definition}

\section{Visible occurrence の個数上界 $V$}
\label{sec:V}

入力の constructor 総数 $\sum_{i=1}^{n}c(x_i)$，目標は $c(y)$，
二項規則はちょうど $n-1$ 回・各回の principal は高々3個なので，
\[
  V = \sum_{i=1}^{n}c(x_i) + c(y) + 3(n-1).
\]
【Lean: \lean{visibleBound}, \lean{visibleSlotsComplete},
\lean{visibleOccurrenceCountBound}】

\section{Invisible trace piece と trace degree}
\label{sec:piece}

trace graph から visible occurrence をすべて取り除き，
残ったグラフの連結成分を \textbf{invisible trace piece} と呼ぶ．
piece $P$ が \textbf{visible boundary} を持つとは，
ある visible occurrence $u$ と $P$ 内の occurrence $o$ の間に
trace edge があることをいう．

各 occurrence の trace degree（trace graph における次数）は
規則集合 $\R$ のみから決まる有限定数 $r$ で上から抑えられる．
Lean 形式化では明示的な近傍列挙により $r = 64$ が証明されている
【Lean: \lean{traceDegreeLimit}】．

\begin{verifybox}
\textbf{【検証：例3 の導出木と trace graph】}

導出：$n_1=a$（葉）$\xRightarrow{T{>}}$ $n_2=s/(a\bs s)$ $\xRightarrow{T{>}}$
$n_3=s/((s/(a\bs s))\bs s)$（深さ4），$n_3,n_4\xRightarrow{>}n_5=s$（根），
$n_4=(s/(a\bs s))\bs s$（葉）．

  \hypo{(s/(a\bs s))\bs s}
  \infer2[$(>)$]{s}
\end{prooftree}
\end{center}

\part{境界なし piece の消去}

\section{目標となる補題と二分割}

\begin{definition}
$\Sig$-導出（途中の全原子が $\Sig$ に属する導出）のうち
constructor occurrence 総数 $\mu(D)$ が最小のものを $\Dmin$ と書く．
\end{definition}

本部の目標は次である：

\begin{quote}
\textbf{境界消去}：$\Dmin$ のすべての invisible trace piece は
visible boundary を持つ．
\end{quote}

旧稿はこれを一様原子置換で示した（旧補題 8）：
「boundary を持たない piece $P$ の occurrence を含む最小の部分範疇を
各メタ変数コピーで一様に固定原子 $p_\ast$ に置き換えれば，
全規則が保存され $\mu$ が真に減る」．
しかしこの論法は\textbf{そのままでは成立しない}：
置換の対象が型上げ骨格 constructor を含む場合，
型上げインスタンス $A \Rightarrow C/(A\bs C)$ の出力の形が置換で壊れ，
unary 規則としての正しさが保てないからである．
そこで piece を二種に分ける：

\begin{definition}
invisible trace piece $P$ が\textbf{置換可能}であるとは，
$P$ が型上げ骨格 constructor の occurrence を含まないことをいう．
そうでない $P$ を\textbf{保護骨格 piece} と呼ぶ．
\end{definition}

\section{置換可能 piece の消去（証明済み）}

\begin{lemma}[置換可能 case]
\label{lem:replaceable}
$D$ を $\Sig$-導出，$P$ を visible boundary を持たない置換可能 piece とする．
このとき同じ入力列・同じ目標をもつ $\Sig$-導出 $D'$ で
$\mu(D') < \mu(D)$ となるものが存在する．
【Lean 証明済み:
\lean{boundaryFreeReplaceableInvisiblePieceContractsFromPieces}】
\end{lemma}

\begin{proof}[証明の概略]
$P$ は visible boundary を持たないので，$P$ の occurrence は
葉・根・principal のいずれにも trace edge で接続しない．
$P$ の各 occurrence の指す部分範疇は補題~\ref{lem:same-sub} により共通の
範疇 $Z$ であり，各節点範疇の中で $P$ が占める位置の族は
trace 閉包により全メタ変数コピーで整合的である．
この族の各位置で部分範疇を一様に $p_\ast \in \Sig$ に置き換える
（置換族，Lean では \lean{PieceReplacementFamily}）．
$P$ が型上げ骨格を含まないから，どの規則インスタンスも
メタ変数コピーの内部だけが書き換わり，規則スキーマは保存される．
$Z$ は constructor を少なくとも一つ含むから $\mu$ は真に減る．
\end{proof}

\section{保護骨格 piece：反例と未解決補題}
\label{sec:protected}

\begin{verifybox}
\textbf{【反例：一様置換が保護骨格 piece で破綻する】}

導出 $D$：入力 $C/X,\;X$，目標 $C$ を
\[
\begin{prooftree}
  \hypo{C/X}
  \infer1[$(T{>},\,C)$]{C/\bigl((C/X)\bs C\bigr)}
  \hypo{X}
  \infer1[$(T{<},\,C)$]{(C/X)\bs C}
  \infer2[$(>)$]{C}
\end{prooftree}
\]
と導く（冗長な導出）．左の $T{>}$ 出力の\textbf{内側骨格} $\bs$
（位置 $1$）と，右の $T{<}$ 出力の\textbf{外側骨格} $\bs$（位置 $\epsilon$）は，
$>$ のメタ変数 $B = (C/X)\bs C$ の二コピーの同じ相対位置として
trace edge で結ばれる．
両者とも葉でも根でも principal でもない（$>$ の principal は左前提の根 $/$ のみ）
から invisible であり，しかもこの2点から他への trace edge はないので，
$P = \{(\text{左},1), (\text{右},\epsilon)\}$ は
\textbf{visible boundary を持たない invisible piece} である．

このとき $P$ を含む部分範疇 $(C/X)\bs C$ を一様に $p_\ast$ に置き換えると，
右前提の unary ステップは「$X \Rightarrow p_\ast$」となり，
$T{<}$ のインスタンスではなくなる．\textbf{一様原子置換は使えない}．

正しい縮約は\textbf{導出の外科手術}である：この形では
\[
  \bigl(T{>}(u) ,\, T{<}(w)\bigr) \xrightarrow{\;>\;} C
  \quad\leadsto\quad
  (u, w) \xrightarrow{\;>\;} C
\]
（$u : C/X$，$w : X$）と型上げ対を丸ごと消去すれば，
同じ入力・同じ目標のより小さい導出が得られる
（Lean の書き換え関係 \lean{Contracts} の \lean{collapseRight} 冗長対）．
この反例は Lean 内で形式的に構成・検証済みである
【Lean: \lean{ProtectedSkeletonCounterexample}】．
\end{verifybox}

一般には，型上げ出力が application で直接消費されるとは限らず，
合成の鎖を経てから相殺されうるため，
上のような局所書き換えだけで尽きるかどうかは自明でない．

\medskip
\noindent\textbf{構造帰納法による帰着（Lean 証明済み）．}
導出木の構造帰納法により，保護骨格 case は大幅に絞り込める：
\begin{itemize}
  \item \textbf{葉}：葉の occurrence はすべて visible なので piece 自体が存在しない．
  \item \textbf{単項根}（根が型上げ）：piece は根の型上げの前提根に触れられない
    （前提根の occurrence は $A$-メタ変数コピーを通じて木の根＝visible に
    trace 接続するため，boundary-free に矛盾）．
    よって piece は前提部分木に制限され，帰納法の仮定が適用できる．
    【Lean: \lean{InvisiblePiece.not\_premiseRoot\_of\_typeRaiseRight},
    \lean{restrict\_typeRaiseRight}】
  \item \textbf{二項根・非横断}（piece がどちらの前提根にも触れない）：
    局所 trace edge の端点は節点アドレス長 $\le 1$ に限られるので，
    前提の内部に真に含まれる piece は辺で反対側に渡れず，
    片側の前提部分木に制限され，帰納法の仮定が適用できる．
    【Lean: \lean{restrict\_binaryLeft}, \lean{restrict\_binaryRight}】
  \item \textbf{二項根・横断}（piece が前提根に触れる）：これだけが残る．
\end{itemize}
【Lean: \lean{sizeReduction\_of\_boundaryFree\_skeleton\_piece}】

\medskip
\noindent\textbf{横断ケースの結論は collapse 冗長対の存在ではない．}
横断ケースの結論を「$D$ が collapse 冗長対（$T{>}$/$>$ または $T{<}$/$<$ の
直接相殺）を含む」と定式化することは\textbf{できない}．次の反例がある．

\begin{verifybox}
\textbf{【反例：合成を挟んだ相殺は collapse 冗長対を持たない】}
$A_0 = C_0/A'$ とし，$w_1 : C/C_0$，$u : A_0$，$u_2 : A'$ から
\[
\begin{prooftree}
  \hypo{w_1 : C/C_0}
  \hypo{u : A_0}
  \infer1[$(T{>},C_0)$]{C_0/(A_0\bs C_0)}
  \infer2[$(B{>})$]{C/(A_0\bs C_0)}
  \hypo{u_2 : A'}
  \infer1[$(T{<},C_0)$]{(C_0/A')\bs C_0 = A_0\bs C_0}
  \infer2[$(>)$]{C}
\end{prooftree}
\]
を導く．三つの occurrence
（$T{>}$ の内側骨格 $([\mathrm{LR}],[1])$，
合成後の functor の引数スロットの $\bs$ $([\mathrm{L}],[1])$，
$T{<}$ の外側骨格 $([\mathrm{R}],\epsilon)$）は，
辺 $B{>}$-$A$ コピーと $>$-$B$ コピーのみで互いに結ばれ，
全 trace 近傍を検証するとこの三点で閉じる——すなわち
\textbf{境界なし・横断・保護骨格含有の piece} である．
しかし $T{>}$ は application ではなく合成に食われているため，
木のどこにも collapse 冗長対（$T{>}$/$>$，$T{<}$/$<$ の直結）は存在しない．
より小さい導出 $w_1\,(u\,u_2)$（app 二回）は存在するが，
そこへ到達するには raised 層を\textbf{三節点で同時に}削る必要がある——
これがまさに band 縮約が要求される理由である．
\end{verifybox}

したがって横断ケースの正しい結論は
「同じ入力列・同じ目標のより小さい（かつ新原子を導入しない）導出の存在」
（Lean: \lean{SizeReduction}）である．以下が残る唯一の未解決補題である．

\begin{conjecture}[保護骨格 case・横断ケース]
\label{conj:protected}
$D$ の根が二項規則で，visible boundary を持たない保護骨格 piece が
いずれかの前提根の occurrence を含むとする．
このとき同じ入力列・同じ目標をもつ導出 $D'$ で
$\mu(D') < \mu(D)$ かつ新原子を導入しないものが存在する．
【Lean 目標: \lean{CrossingBoundaryFreeSkeletonPieceReduces}】
\end{conjecture}

\medskip
\noindent\textbf{証明済みの部分ケース：合成スパイン．}
反例を含む族——「型上げで終わる合成スパイン」——については
未解決補題~\ref{conj:protected} の結論が Lean で証明済みである：
$F = x_1 \circ (x_2 \circ (\cdots \circ T{>}_{C_0}(u)))$（$B{>}$ の鎖）を
raised スロット $A_0\bs C_0$ の引数 $z$ に適用した
$F\,z$ は，application の連鎖
$x_1\,(x_2\,(\cdots\,(u \mathbin{<} z)))$ に縮約でき，
サイズは少なくとも $c(A_0\bs C_0)+1$ 減り，新原子を導入しない
（スパインの各節点から raised 層が同時に消える——band 削除の
スパイン特殊ケース）．鏡像（$B{<}$ の鎖と $T{<}$，$<$ での消費）も同様．
【Lean: \lean{CompSpineRight.collapse},
\lean{sizeReduction\_appRight\_of\_compSpineRight},
\lean{CompSpineLeft.collapse},
\lean{sizeReduction\_appLeft\_of\_compSpineLeft}】
スパイン長 $0$ は collapse 冗長対そのものである．
残る内容は，「境界なし横断 piece の存在」から
「相殺構造（一般には合成スパインとは限らない）の存在」を導き，
その一般の相殺構造に対して band 削除を構成する部分である．

\medskip
\noindent\textbf{横断 piece の構造（application の場合，Lean 証明済み）．}
根の規則が application（$>$ または $<$）のとき，横断 piece は
消去される引数範疇 $B$ の内部にしか入れない：
functor 前提の結果部（$C/B$ の $C$ 側）の occurrence は
根（visible）へ trace 接続するため boundary-free に反し，
前提根の位置 $\epsilon$ は principal（visible）である．
さらに横断 occurrence のメタ変数パートナー
（$>$ の $B$-コピー対 $([\mathrm{L}], 1{\cdot}q) \leftrightarrow ([\mathrm{R}], q)$）
は必ず piece に属する．
帰結として：(i) 消去される側の前提が\textbf{葉}なら横断 piece は存在しない
（葉の occurrence は visible）；
(ii) functor 前提が\textbf{型上げ}なら根が collapse 冗長対（スパイン長 $0$）．
すなわち application の横断ケースが未解決なのは
\textbf{functor 前提が二項規則で作られている場合のみ}である．
【Lean: \lean{appRight\_left\_in\_argument},
\lean{appRight\_partner\_of\_\{left,right\}},
\lean{crossing\_appRight\_functor\_leaf\_false} と各鏡像】

\medskip
\noindent\textbf{合成規則の場合（Lean 証明済み）．}
根の規則が合成（$B{>}$，$B{<}$，$Bx{>}$，$Bx{<}$）のときも同様に，
横断 piece は消去される中間範疇 $B$ の内部に閉じ込められ，
$B$-メタ変数コピーの対
（$B{>}$: $([\mathrm{L}],1{\cdot}p)\leftrightarrow([\mathrm{R}],0{\cdot}p)$；
$B{<}$: 同型；$Bx{>}$: $1{\cdot}p\leftrightarrow 1{\cdot}p$；
$Bx{<}$: $0{\cdot}p\leftrightarrow 0{\cdot}p$）が必ず piece に属する
【Lean: \lean{crossing\_\{compRight,compLeft,crossedRight,crossedLeft\}\_pair}】．

\medskip
\noindent\textbf{部分木アドレス化（Lean 証明済み）．}
一般の追跡（piece を前提の内部へ辿る）のための基盤として，
部分木の occurrence と trace 辺を任意深さのアドレスに沿って全体の木へ
持ち上げる機構を形式化した
【Lean: \lean{SubtreeAt}, \lean{SubtreeAt.lift\_traceEdge},
\lean{SubtreeAt.trans}, \lean{SubtreeAt.isLeafNode\_of\_leaf}】．

\begin{lemma}[境界消去]
\label{lem:boundary}
未解決補題~\ref{conj:protected} を仮定する．
このとき $\Dmin$ のすべての invisible trace piece は visible boundary を持つ．
【Lean: \lean{boundaryFreeInvisiblePieceElimination\_of\_replaceable\_or\_protected\_piece\_contracts}】
\end{lemma}

\begin{proof}
boundary を持たない piece があれば，置換可能なら補題~\ref{lem:replaceable}，
保護骨格 piece なら未解決補題~\ref{conj:protected} により
$\mu$ が真に減る導出が得られ，$\Dmin$ の最小性に反する．
\end{proof}

\section{深さ上界 $H = V + rV$ の数え上げ}
\label{sec:depth}

\begin{theorem}[branch 数え上げ]
\label{thm:count}
$D$ を導出図とし，$D$ のすべての invisible trace piece が
visible boundary を持つとする．
このとき任意の節点範疇 $A_v$ について
\[
  d(A_v) \le V + rV =: H(\Gamma, y).
\]
【Lean 証明済み: \lean{nodeConstructorPositionsBounded\_of\_boundary\_trace}】
\end{theorem}

\begin{proof}
$A_v$ の根から constructor position $p$ までの branch を固定する．
branch 上の occurrence 数は $|p|+1 = $（その branch の深さ）である．
branch 上の occurrence を visible と invisible に分ける．

\medskip
\noindent\textbf{Visible．} 全導出で visible occurrence は高々 $V$ 個
（\S\ref{sec:V}）なので，branch 上にも高々 $V$ 個．

\medskip
\noindent\textbf{Invisible．}
invisible occurrence $o$ の属する piece は仮定により visible boundary を持つ：
すなわちある invisible な代表 $o'$（$o$ と同じ piece）と
visible occurrence $u$ が存在して $o' \sim u$ が trace edge．
そこで $o$ に符号
\[
  \mathrm{code}(o) = (\,u \text{ の visible slot 番号} \le V,\;\;
  o' \text{ が } u \text{ の近傍リスト中で占める位置} \le r\,)
\]
を割り当てる（trace degree $\le r = 64$）．
符号が一致する二つの invisible occurrence $o_1, o_2$ は
同じ $u$・同じ $o'$ を持つから同じ piece に属する．
ところが同一 branch 上の異なる二点は系~\ref{cor:no-revisit} により
同じ piece に属せない．ゆえに $\mathrm{code}$ は branch 上で単射であり，
invisible occurrence は高々 $rV$ 個．

\medskip
\noindent\textbf{合計．} $d(A_v) \le V + rV$．
\end{proof}

\begin{remark}[旧稿の上界との比較]
旧稿の上界は $H = V + qrV(V+1)$（$q$ は interface state 数 $\approx 2500$）
であった．interface state による細分と visible-free zone への分割は，
repeatable pair の禁止（空虚）を経由するための道具立てだったが，
系~\ref{cor:no-revisit} が「piece の一致」だけで branch 上の重複を禁止するため，
どちらも不要である．$r = 64$ のとき $H = 65V$ であり，
上界は $V$ について線形になる．これは chart parsing の候補集合
$K_{\Gamma,y}$ の深さパラメータを現実的な規模に近づける．
\end{remark}

\section{有界正規形定理}

\begin{theorem}[Bounded-category normal form]
\label{thm:bounded}
未解決補題~\ref{conj:protected} を仮定する．
$x_1,\ldots,x_n\Rightarrow_{\R}^{*}y$ ならば，
途中の全範疇 $A$ について $\At(A)\subseteq\Sig$ かつ $d(A)\le H(\Gamma,y)$
を満たす同じ入力列・目標の導出 $D'$ が存在する．
【Lean: \lean{depthBoundedNormalForm\_of\_replaceable\_or\_protected\_piece\_contracts\_and\_trace}】
\end{theorem}

\begin{proof}
補題~\ref{lem:atom} より $\Sig$-導出があり，その $\mu$-最小のもの $\Dmin$ をとる．
補題~\ref{lem:boundary} より $\Dmin$ の全 piece が visible boundary を持ち，
定理~\ref{thm:count} より全節点範疇の深さは $H$ 以下．$D' = \Dmin$．
\end{proof}

\section{有限候補範疇集合}

\[
  K_{\Gamma,y} = \{\,A\mid \At(A)\subseteq\Sig,\;d(A)\le H(\Gamma,y)\,\}.
\]
$\Sig$ が有限・深さ有界なので $K_{\Gamma,y}$ は有限かつ機械的に列挙可能
【Lean: \lean{categoryPool}, \lean{mem\_categoryPool\_of\_atoms\_depth}】．

\section{有限 chart algorithm}
\label{sec:chart-alg}

各区間 $[i,j)$ に有限集合 $\Chart[i,j]\subseteq K_{\Gamma,y}$ を持つ chart を作り，
以下の closure を固定点まで繰り返す．

\medskip
\noindent\textbf{Unary closure．}
$A\in\Chart[i,j]$ かつ $C/(A\bs C)\in K_{\Gamma,y}$ なら追加；
$(C/A)\bs C\in K_{\Gamma,y}$ なら同様．

\medskip
\noindent\textbf{Binary closure．}
$A\in\Chart[i,k]$，$B\in\Chart[k,j]$，六つの二項規則のいずれかで
$C\in K_{\Gamma,y}$ が得られれば追加．

\begin{verifybox}
\textbf{【例：例1 の chart trace】}
入力 $a,\;(a\bs s)/b,\;b$（$n=3$），目標 $s$：
$\Chart[0,1]$ で unary closure が $T{>}(a){=}s/(a\bs s)$ を追加，
$B{>}$ で $\Chart[0,2]\ni s/b$，$>$ で $\Chart[0,3]\ni s$．
導出可能性が chart 上で確認される．
\end{verifybox}

\section{停止性・健全性・完全性}

chart item 総数は有限なので closure は必ず停止する（停止性）．
item 追加順の帰納法により，chart に入る item は実際に導出可能
（健全性；\textbf{無条件に} Lean 証明済み:
\lean{completeBoundParser\_sound}）．
さらに，chart の受理は「全中間範疇が $K_{\Gamma,y}$ に属する導出の存在」と
\textbf{正確に}一致する（有界関係の正確性；無条件に Lean 証明済み:
\lean{recognize\_sound\_chartDerivable\_candidate}）．
定理~\ref{thm:bounded}（未解決補題~\ref{conj:protected} に条件付き）より
成功導出は $K_{\Gamma,y}$ 内の導出を持つので，完全性が従う
【Lean: \lean{completeBoundParserComplete\_of\_protectedSkeletonPieceContracts}】．

\section{決定可能性}

健全性と完全性より，
\[
  x_1,\ldots,x_n\Rightarrow_{\R}^{*}y
  \quad\Longleftrightarrow\quad
  y\in\Chart[0,n].
\]
右辺は有限集合上の固定点計算で判定でき，停止する．

  enhanced, colback=white, colframe=black,

未解決補題~\ref{conj:protected} を仮定すれば，
\[
  x_1,\ldots,x_n
  \Rightarrow_{\{<,>,T{<},T{>},B{<},B{>},Bx{<},Bx{>}\}}^{*}
  y
  \quad\text{は判定可能である．}
\]

\section*{証明の要点}
\addcontentsline{toc}{section}{証明の要点}
%% ============================================================

第一に，入力と目標に現れない原子は準同型で潰せる（補題~\ref{lem:atom}）．

第二に，trace edge は指し先の部分範疇を保存する（補題~\ref{lem:same-sub}）．
この一つの観察から，(a) 旧稿の repeatable pair / band contraction 機構が
充足不能ゆえ不要であること，(b) 一つの invisible piece が
一本の branch と高々一度しか交わらないこと（系~\ref{cor:no-revisit}）が同時に従う．

第三に，$\mu$-最小導出において境界なし invisible piece は存在しない
（補題~\ref{lem:boundary}）——ただし piece が型上げ骨格を含む場合の縮約
（未解決補題~\ref{conj:protected}）が残る唯一のギャップである．
型上げ骨格を含まない場合は一様原子置換で Lean 証明済み（補題~\ref{lem:replaceable}）．

第四に，境界条件の下で中間範疇の深さは
$H(\Gamma,y) = V + rV$（$r = 64$）で有界（定理~\ref{thm:count}）．
invisible occurrence は（boundary の visible slot, 近傍 slot）の対で
branch 上単射に符号化される．

第五に，深さ有界より有限 chart parsing が停止し，健全かつ
（未解決補題に条件付きで）完全である．
